# X2Nitter

Convert Twitter/X links to [Nitter](https://github.com/zedeus/nitter) with one click.

Paste a `twitter.com` or `x.com` link, hit Open, and it loads on nitter.net (or pick another instance from the dropdown).

Static single-file app. No backend, no tracking, no dependencies.

**[Try it live](https://nschumac.github.io/X2Nitter/)**
